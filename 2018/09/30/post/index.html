Notes about PHP

Posted on 2018-09-30 | In study

This is the note what i learned from PHP.

第一天

php:服务器端的开发语言。

php能完成的工作：收集表单数据；生成动态网页（主要功能）；字符串处理；动态输出图像；服务器端文件系统操作；编写数据库支持的网页（php可以连接任何世界标准的数据库）；
会话跟踪控制；处理XML文件；支持利用大量的网络协议（也可以开放原始网络端口，使得任何网络协议都可以在上面工作）；服务器端的其他相关操作；所有的主流操作系统都可以搭
载，大部分的web服务器也可以（如：apache,iis等）

CGI：通用网关接口。
oop（面向对象编程）：最符合程序员的开发思想。

在php文件中可以编写html,css,javascript等语言，但使用起始标记<?php和?>之前的代码用php应用服务器解析，将解析后的代码发到客户浏览器，在这个标记之外的，php服务器都
不解析，直接发送给客户端，客户端再用浏览器解析。

在整个程序的任何地方，都可以有起始和结束标记的使用。

<?php ?> 这种标记，推荐使用，php程序员无法禁用。
<?  ?> 这种标记，php程序员有时会禁用，因为它会干扰XML的文档声明。
<%  %> 这种标记，移植性较差。
<script language="php"> </script> 这种标记，不推荐，无法插入html文件当中。

语句：功能执行语句（其后一定要加分号）（与?>最近的一条语句可以不加分号，建议都加分号）
      结构定义语句（其后一定不要加分号）

// 单行注释；/* */ 多行注释（其中不能再包含多行注释，但可以包含单行）；# 文档注释；/** */ 文档注释
注释要写在代码的上面或者右边。

如果?>标记之后，就是脚本结束，之后没有php程序了，可以不加该标记（建议不加）

第二天

变量：是值在程序的运行过程中可以随时发生变化的量，是程序中数据的临时存放场所。
Isset(); 判断变量是否存在（输出0或1）
unset(); 释放变量
"$" 变量的声明和使用都要有这个符号。

php可以使用系统关键字作为变量名。（如if,for,switch等）
注意：php变量区分大小写。（只有变量和常量区分大小写）
变量名称命名一定要有意义。
php里面的变量赋值一般是传值赋值。
变量的地址通常是十六进制。
存地址的变量就是指针。

php中共有8种类型：4种标量（整形，布尔型，浮点型，字符串），2种复合类型（数组array，对象object），2种特殊类型（资源类型resource，空类型null）。
Var_dump(变量和值) 输出它的值和类型。
以0开头的表示以8进制声明一个变量，以0x或0X开头的表示以16进制声明一个变量，整数的最大值是4个字节，
3.14E5 = 3.14E+5 = 3.14*(10^5)  
布尔值是false的情况，零，null，零字符串，空字符串，无成员的数组。

字符串声明的方法：1.单引号和双引号都可以声明字符串。2.声明的字符串没有长度限制。3.在双引号的字符串中，既可以直接解析变量，又可以直接使用转义字符。
4.在单引号的字符串中，不可以解析变量，也不可以使用转义字符。5.在双引号中不能再使用双引号，在单引号中不能再使用单引号。5.最好使用单引号。（双引号会更
多的使用资源）

数据类型之间的相互转换：强制转换（setType(变量,类型)；在赋值前使用(类型)的形式，不会改变原变量的类型），自动转换（常用，开发时变量会根据运行环境自动转换）。

与变量和类型有关的一些变量和函数：
issset(); 判断一个变量是否存在（输出1或0）
empty(); 判断一个变量是否为空（输出1或0）
unset(); 释放一个变量
getType(); 得到一个变量的类型

常量的声明与使用：
1.常量是一个简单值的标识符。
2.不能使用unset()取消。
3.常量可以不用理会变量范围的规则，而在任何地方都可以定义和访问。
4.常量使用define("常量名","值");
5.常量声明和使用都不使用"$"。
6.常量名成习惯都使用大写。
7.常量只能用做标量（int,float,bool,string）
8.defined("常量");  

预定义常量（使用指定的名称就会输出里面的值）和魔术常量（会随不同的情况而改变）

运算符号：按运算符的功能分为：算数运算符（+ - * / % ++ --），字符串运算符（.(可以叫做连接运算符)），赋值运算符（= += -= *= /= %= .=(连接)），比较运算符（> 
< >= <= == ===(比较是内容和类型都要相同) != <> !==）（也叫条件运算符，关系运算符），逻辑运算符（&&或and ||或or !=或<> !==）（只能操作bool型值，返回的也是
bool型值），位运算符（&& | ^ ~ << >> >>>），其他运算符（？ : `` @ => -> :: & $）

短路问题：&&和||会发生短路。（&&在做运算时，若前面的数是false，则后面的数是否位true，不再去判断）（||在做运算时，若前面的数为true，则后面的是否位false，不
再去判断）（&或|在做运算时，两边都会被执行）

优先级：最好是用括号来改变优先级。
表达式：只要有变量和运算符参与的式子就是表达式。
